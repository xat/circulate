# circulate

circulate is a function for endless looping over an array.

```javascript
var next = circulate([1, 2, 3]);

next(); // 1
next(); // 2
next(); // 3
next(); // 1
next(); // 2

// ... and so on
```


## Installation

### Browser

```bower install circulate```
